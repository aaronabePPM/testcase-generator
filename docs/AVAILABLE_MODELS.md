## üîß Troubleshooting

### "Model not available"
- Check your API key has access to the model
- For GitHub Models, ensure you're using a valid token
- Try a different model from the list

### "Rate limit exceeded"
- GitHub Models have rate limits during preview
- Wait a few minutes and try again
- Consider using Anthropic or OpenAI direct

### "Invalid API key"
- Verify you're using the correct key for the provider
- GitHub Models uses GitHub personal access tokens
- Anthropic uses API keys from console.anthropic.com
- OpenAI uses keys from platform.openai.com
